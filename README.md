## Problem Statement
Phishing attacks are one of the most common cybersecurity threats, leading to financial loss, data breaches, and identity theft. Traditional rule-based systems fail to detect evolving phishing patterns. This project aims to build a scalable and intelligent solution that can accurately classify phishing and legitimate instances.
